# Apartnetwork

Apartnetwork — учебно-прикладное desktop-приложение на `C++` и `Qt`, предназначенное для интеллектуальной оценки стоимости квартир в Москве. Проект объединяет собственную нейросетевую модель, пользовательский интерфейс на `QML`, аналитические визуализации и воспроизводимую выборку, откалиброванную под рыночные диапазоны 2023 года.

Это не просто форма с числовым ответом, а цельная демонстрация полного прикладного цикла: от подготовки данных и обучения модели до интерфейса, тестов, документации и интерпретации результата. Приложение показывает, как методы машинного обучения могут быть встроены в настольный продукт для решения реальной задачи оценки недвижимости.

## Краткое описание

Проект решает задачу ориентировочной оценки стоимости квартиры по набору пользовательских параметров:

- площадь квартиры;
- количество комнат;
- этаж и этажность дома;
- удалённость от метро;
- округ Москвы;
- год постройки;
- тип дома;
- состояние квартиры;
- наличие паркинга, балкона и признака нового дома.

На основе этих характеристик приложение формирует оценку стоимости объекта, выводит ориентир по цене за квадратный метр и сопровождает результат аналитическими элементами, которые помогают понять контекст расчёта.

## Практическая ценность проекта

- демонстрирует интеграцию ML-модели в полноценное desktop-приложение;
- показывает связку `данные + модель + интерфейс + тесты + документация` в одном проекте;
- может использоваться как учебный, курсовой, демонстрационный или портфолио-проект;
- подходит как пример того, как адаптировать технический прототип под конкретную предметную область.

## Что делает проект сильнее обычного прототипа

- модель работает на выборке, приведённой к задаче оценки квартир Москвы 2023 года;
- интерфейс не ограничивается одной кнопкой прогноза, а включает визуализацию выборки и процесса обучения;
- в проект добавлены unit-тесты, что делает его более инженерно завершённым;
- документация описывает не только запуск, но и устройство датасета, логику признаков и ограничения модели;
- структура проекта разделяет модель, адаптер, viewmodel, UI и тесты, что упрощает развитие и поддержку.

## Ключевые возможности

- оценка стоимости квартиры по параметрам пользователя;
- вывод ориентировочной полной цены и цены за квадратный метр;
- визуализация встроенной выборки квартир Москвы;
- отображение динамики обучения модели по метрике `MSE`;
- быстрые сценарии через пресеты;
- воспроизводимая генерация датасета;
- локальные unit-тесты для ключевой логики.

## Что доработано

- выборка переведена со старого датасета домов на модель квартир Москвы 2023 года;
- форма прогноза переделана под квартирные признаки: округ, этаж, год постройки, расстояние до метро, тип дома, состояние, новый дом, паркинг, балкон;
- добавлены визуализации:
  - scatter chart по выборке;
  - кривая обучения по `MSE`;
  - карточки ключевых метрик;
  - окружные коэффициенты внутри выборки;
- добавлены unit-тесты на парсинг `CSV`, обучение модели и сравнительный прогноз;
- добавлена документация по запуску и по устройству выборки.

## Сборка

Требования:

- `Qt 6.2+` с компонентами `Core`, `Gui`, `Qml`, `Quick`, `Concurrent`, `Test`
- `CMake 3.21+`
- рабочий `C++` toolchain

Пример:

```bash
cmake -S . -B build -G Ninja -DBUILD_QDS_COMPONENTS=OFF
cmake --build build
```

`BUILD_QDS_COMPONENTS` по умолчанию отключён, чтобы проект не тянул внешние `Qt Design Studio`-компоненты при обычной сборке.

## Тесты

```bash
cmake -S . -B build -G Ninja -DBUILD_QDS_COMPONENTS=OFF
cmake --build build --target ApartnetworkTests
ctest --test-dir build --output-on-failure
```

## О выборке

Выборка не является официальной выгрузкой из Росреестра или ЦИАН.  
Это синтетический датасет, сгенерированный скриптом и откалиброванный под публичные рыночные ориентиры Москвы за 2023 год.

Внутри выборки учитываются массовый сегмент, более дорогие округа, новые дома и качественные различия объектов, поэтому приложение работает не на абстрактных числах, а на доменной модели, приближённой к реальной задаче.

- вторичный рынок старой Москвы в конце 2023 года находился около `270 тыс. ₽/м²`;
- массовый сегмент новостроек Москвы по итогам декабря 2023 года находился около `326 тыс. ₽/м²`;
- дорогие округа и новые дома внутри выборки получают отдельные коэффициенты.

Подробности: [docs/DATASET.md](docs/DATASET.md)

## Ограничения

- модель учебная и даёт ориентировочную, а не юридически значимую оценку;
- выборка синтетическая, поэтому результат лучше использовать как диапазонный ориентир;
- для полноценного production-сценария нужна живая выгрузка объявлений или сделок и отдельная валидация на holdout-наборе.
